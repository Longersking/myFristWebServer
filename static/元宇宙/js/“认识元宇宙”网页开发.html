元宇宙的概念

对于元宇宙的畅想,始于1992年出版的科幻小说《雪崩》。书中描述了一个名为Metaverse(元宇宙)的虚拟世界，人们只要通过公共入口连接，就能以虚拟“化身”的形象进入其中，开启有别于现实世界的另一种人生。近年来，随着全球科技进步，元宇宙的内涵外延不断演进。 元宇宙一般指是一个虚拟时空间的集合,这个词通常用来描述未来互联网迭代的概念,由连接到一个可感知的虚拟宇宙的持久、共享的3D 虚拟空间组成。广义上的元宇宙不仅指虚拟世界，还指整个互联网，包括增强现实的整个范围。

元宇宙的未来发展趋势

其实现在所构建出的元宇宙是为娱乐以及社交而服务，同时很多资本向此靠拢，这也是元宇宙发展趋势的一大方向--也就是消费市场，也是现在作为主流形态的游戏领域，侧重于消费体验以及娱乐社交。

元宇宙在未来的商业市场的潜力不容小觑。它可能会将现在的智慧生活构建提升一个等级，在智慧制造、智慧家居、智慧办公等等上改变人们最根本的生活工作方式。实现事半功倍。

联想在元宇宙初步应用到企业研发制造场景上走在了前列。在智能制造转型的探索中，联想孕育出了5大能力：：互联互通、柔性制造、虚实结合、闭环质量、智能决策。

元宇宙引出了“数字孪生”。数字孪生是一种超越现实的概念，可以被视为一个或多个重要的、彼此依赖的装备系统的数字映射系统。

各国对元宇宙的态度与政策

各个国家对元宇宙进行了哪些探索？又出台了哪些适用于元宇宙的政策法规？

尽管国内外目前正在积极探索元宇宙，元宇宙还是一个非常新的概念，几乎所有的国家还没有出台相关的政策法规。

  • 美国

    美国率先提出“元界”概念，监管关注数据安全。美国是元宇宙领域的先行者，而且美国的监管机构重点关注数据安全和隐私保护问题，如生物特征、位置和银行信息、消费习惯、游戏习惯等，都属于数据安全和隐私保护的范畴。

  • 韩国

    韩国政府对元宇宙反应最快，已经成立了元宇宙协会。韩国首都首尔市政府宣布，将打造“元宇宙平台”，向市民提供公共服务。韩国政府也在探索将元宇宙应用到旅游业的方式。

  • 俄罗斯

    普京为元宇宙发展指明方向，并表达了直面挑战的决心。俄罗斯现任总统普京认为，元宇宙的价值在于让人们不论相距多远，都可以一起交流、工作、学习、落实联合创新项目和商业项目，而不是人们从不完美的现实世界逃离的目的地。

  • 中国

    企业密集布局元宇宙，炒作风险成监管重点。在探索元宇宙的赛道上，我们中国也没有落后。继韩国成立元宇宙协会后，中国移动通信联合会元宇宙产业委员会举办揭牌仪式，标志着国内首家获批元宇宙行业协会正式揭牌。

元宇宙的弊

  • 想象的共性是人性的体现,而其差异性所映射出的同样是人性

    元宇宙可能会沦为恣意宣泄恶念、突破道德底线的虚拟“法外之地”

  • 对人们现实世界中闲暇时间分配利用的影响

    互联网时代的各种“成瘾”已经成为现代社会的顽疾，元宇宙是否会为这一顽疾再添一个瘾性巨大的“病灶”值得讨论和预判

  • 放大“数字鸿沟”

    穿行于物理世界与个人灵境之间靠的是各种先进基础设施和智能硬件，鉴于各国经济发展水平和基础设施以及不同人群之间收入的现实差距，科幻作品《北京折叠》中的场面会在元宇宙下变得更加尖锐，进而加剧社会分化的矛盾

  • 使人不耗费任何成本就享受了精神上的满足

    人类便没有动力去为了实现更美好的生活去不断创新不断创造

元宇宙的发展

人类对元宇宙的设想、构建的过程实际上是人类将自己的价值观念、人文思想、技术工具、经济模式和“宇宙”认知结合的过程。从开端到元宇宙初见成效，经历了三个历史阶段，东西方在时间长河中体现着他们自己的理解。

第一阶段

以艺术、文化为载体的古典形态元宇宙

西方的《神曲》、《天神右翼》等都属于元宇宙，他们通过幻想，创建出给予灵魂归宿的世界，构建闭环式的虚拟宇宙。东方文明中，《易经》《西游记》等，通过创造与价值探索，寻求事物变化规律与存在本质，不断推演，创造出一个基于现实，又伴随着玄幻色彩的世界，赋予着美好万物的价值与未来的无限幻想。

Read More

第二阶段

伴随着互联网与科幻世界的形成与发展构建的新古典元宇宙

随着信息革命的到来，人们逐渐认识到过去的畅想，由于互联网、芯片、存储器的发展已不再是天方夜谭，引发了信息可续、量子科学、数学和生命科学的互动，改变了科学范式，推动了传统哲学、社会学甚至人类科学体系的突破。1996年，通过虚拟现实建模语言（VRML）构建的Cybertown，是新古典“元宇宙”重要的历程碑。这一阶段，人们对元宇宙的畅想不断传播，为真实实现描绘了宏伟蓝图。

Read More

第三阶段

以“去中心化”游戏为载体的高度智能化形态元宇宙

2003年，美国互联网公司推出的Linden Lab推出基于Open3D的“第二人生”是这一阶段的标准性成就，它允许人在除现实世界外拥有第二身份，去过一种理想中的大同社会。2006年，Roblox公司创建Roblox平台，同时兼容了虚拟世界、休闲游戏和自建内容，用户可自行设计游戏作品，覆盖角色扮演、第一人称射击、动作格斗、生存、竞速等玩法，以及其他各种各样的创意内容，拥有极高的自由度。2021年，Facebook改名Meta正式进军元宇宙，扎克伯格展示自己的虚拟形象，进入虚拟VR社交世界、容纳越来越多的用户，将元宇宙建设推向高潮。
